# Readme

## Usage


### add-copyright

```bash
add-copyright {sources_file_or_directories}
```

All files in a given directory

```bash
add-copyright path/to/dir
```

Specific files

```bash
add-copyright file_1 path/to/file_2
```

### Configuration

Can read from a `pyproject.toml` file.
